Layout du dashboard vendeur — 3 approches

Quelle direction pour le redesign complet (onboarding + dashboard) ?

A

Dark — Pleine page

Fond #08080f, cards glass bg-white/[0.03], bordures subtiles. Même univers que la landing. Sidebar collapsible en overlay. Onboarding = flow vertical centré.

B

Clean — Sidebar fixe

Sidebar latérale fixe (260px) avec fond sombre, zone de contenu claire. Onboarding = card centrée. Proche du layout actuel mais modernisé. Sidebar toujours visible.

C

Minimal — Sans sidebar

Pas de sidebar pendant l'onboarding. Plein écran, fond dark avec effet glass. Après inscription, bascule vers le dashboard avec sidebar. Flow onboarding = tunnel vertical.
